# mkmr

Small python3 utility to create Merge Requests on GitLab, with special support for Alpine Linux's self-hosted instance.

## Dependencies

The packages for Alpine Linux, please check the equivalents for your distribution:

* py3-gitpython
* py3-urllib
* py3-gitlab
* py3-python-editor
* py3-inquirer
